# SWING RUSH

## ゲームの内容
爽快感を重視した一人称ワイヤーアクションゲーム

地面に触れずビルの間をすり抜けてゴールを目指せ！

![画面イメージ](docs/images/game_image01.png)

## 操作説明
- WASDでキャラクタを前後左右移動（W→前・A→左・S→後・D→右）
- スペースキーを押すとワイヤーを巻き取って縮めることができる
